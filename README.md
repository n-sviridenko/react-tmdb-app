Getting started
---------------

1. Clone the repository:<br />
`$ git clone https://github.com/n-sviridenko/react-tmdb-app`<br />
and open its folder:<br />
`$ cd react-tmdb-app`

2. Install dependencies:<br />
`$ npm install`<br />
or using:<br />
`$ yarn install`

3. Launch the client:<br />
`$ yarn start`
